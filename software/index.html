Software/Codes

Implementing econometric methods can be challenging and time-consuming. This page provides programs that implement methods and models related to my research. Some of these programs are available as R packages that I have developed. These packages are published on CRAN or GitHub. Loading CRAN stats...

Packages

  • CDatanet - An R Package for the Econometrics of Network Data, first released in February 2021

  • vMF - An R Package for Fast Sampling from the von Mises–Fisher Distribution, first released in November 2022

  • PartialNetwork - An R Package for Estimating Peer Effects Using Partial Network Data (with Vincent Boucher), first released in August 2023

  • QuantilePeer - An R package for Simulating and Estimation Quantile Peer Effect Models, first released in June 2025
